  • Data Center Project for New UC Berkeley Course
    Design optimization combining a genetic algorithm with a 2D finite difference thermal solver. Students would follow a template to write the code for both the solver and the genetic algorithm to determine an optimal layout of chip racks in a data center that maximized total compute given the thermal limits of the chips. Project was handed off to Professor Thomas Schutzius who is currently teaching the course.
